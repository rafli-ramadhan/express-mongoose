# Express + Mongoose + MongoDB (Auth and CRUD Application)

## Requirement

1. Node.js
2. MongoDB

## Set Up :rocket:

```
npm install
```

```
npm run dev
```

App will be running on port 5000 (http://localhost:5000).

## API Endpoint

Open `./routes`
